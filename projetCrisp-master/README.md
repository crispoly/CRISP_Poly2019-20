# projetCRISP

Projet CRISP est un robot humanoide collaboratif.

Le code présent permet de contrôler le robot par un interface MATLAB. Celui-ci communique avec des arduinos,contrôleurs TMCM-1633 via RS485.


Langages: MATLAB-Arduino
